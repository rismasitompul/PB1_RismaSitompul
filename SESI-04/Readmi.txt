------------------

Algoritma 
-> Kumpulan proses ataupun aturan untuk melakukan atau menyelesaikan sesuatu.

Kriteria Algoritma :
  - Ada Input dan Output
  - Efektivitas dan Efisien
  - Terstruktur

Flowchart
---------
-> - Bagan-bagan yang mempunyai arus menggambarkan langkah-langkah penyesuaian suatu masalah.
   - Merupakan cara penyajian dari suatu Algoritma
   - Ada 2 macam Flowchart :
        1. System Flowchart , menunjukkan alat media input, output serta jenis penyimpanan dakam proses pengolahan data
        2. Program Flowchart, urutan instruksi yang digambarkan dengan symbol tertentu untuk memecahkan masalah dalam suatu program.
        

Penulisan Conditional 
----------------------

let kondisi = "lapar";
// MultiLine
    if(kondisi != "lapar") {
        console.log("Iam Happy")
    } else {
        console.log("I Want To Eat");
    }

// Singgle Line
    if(kondisi != "lapar") 
        console.log("Iam Happy");
        else console.log("pengen makan");
